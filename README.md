# RL_TECHNICAL_PROJECT
Repository for the final technical project of the Robotics Lab 2025/26 course
Project developed for the Robotics Lab course at the University Federico II of Naples. The system involves two robots collaborating in an industrial environment simulated in Gazebo: Fra2mo autonomously explores the map and detects an ArUco cube using computer vision (aruco_ros library), while ARMcar — a custom differential robot equipped with a manipulator — navigates toward the target with LiDAR-based obstacle avoidance, and physically presses a button on the cube using its gripper. The entire architecture is built on ROS 2, with dedicated Python nodes handling marker detection, odometry-based goal navigation, reactive obstacle avoidance, and manipulator control with joint state feedback.
